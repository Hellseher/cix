**** nc-170109001313
setting up a one-shot webserver on port 8080 to present the content of a file:
:    ~$ { echo -ne "HTTP/1.0 200 OK\r\nContent-Length: $(wc -c < input_f)\r\n\r\n" cat input_f; } | nc -l -p8080
